# CAD to GIS Documentation Workflow

## Overview
This repository documents workflows for translating architectural drawings
into spatial databases suitable for GIS-based analysis.

## Workflow
1. CAD drawing cleanup and layer logic
2. Attribute definition and data structuring
3. GIS import and georeferencing
4. Spatial querying and visualization

## Applications
- Historic districts
- Built artworks and mosaics
- Condition mapping
- Urban-scale heritage documentation
